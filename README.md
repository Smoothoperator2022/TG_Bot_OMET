# \# TG\_Bot\_OMET

# 

# Telegram bot developed for the Odesa Municipal Electric Transport Museum (OMET).

# 

# \## Features

# 

# \* Registration for museum excursions.

# \* Visitor quantity management.

# \* SQLite database storage.

# \* Feedback system:

# 

# &#x20; \* text messages

# &#x20; \* voice messages

# &#x20; \* images

# \* Social network links.

# \* Telephone contact collection.

# 

# \## Technologies

# 

# \* Python 3

# \* pyTelegramBotAPI

# \* SQLite

# \* requests

# \* python-dotenv

# 

# \## Installation

# 

# ```bash

# git clone https://github.com/Smoothoperator2022/TG\_Bot\_OMET.git

# cd TG\_Bot\_OMET

# 

# python -m venv venv

# 

# venv\\Scripts\\activate

# 

# pip install -r requirements.txt

# ```

# 

# Create `.env` file:

# 

# ```env

# BOT\_TOKEN=your\_token\_here

# ```

# 

# Run:

# 

# ```bash

# python OMET\_bot.py

# ```

#
